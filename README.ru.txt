IL2 stats - система статистики для выделенного сервера Ил-2 Битва за Сталинград.
Предназначена для сбора информации о действиях игроков и систематизации этих данных на конкретном выделенном сервере.

Данное ПО задумано, разработано и поддерживается командой из двух человек (=FB=Vaal и =FB=Isay) исключительно как личная инициатива, т.е. не является проектом студии 1CGS.
Основная мотивация для создания данного ПО - желание развивать проект Ил-2 БЗС,
создать новые возможности для комьюнити в организации мультиплеерных проектов Ил-2 БЗС.
ПО распространяется бесплатно. Лицензия MIT.
Авторы ПО не дают ни каких гарантий и не несут ни какой ответственности.


!!!!! ВАЖНО !!!!!

Алгоритмы сбора статистики IL2 stats отличаются от статистики в игре. Как следствие данная статистика не будет совпадать с игровой.
Система учета сбитых рассчитана на сервера с активированной настройкой - завершение вылета только на земле.

===== ОПЦИИ =====

Название сервера, отображаемое на сайте изменяется в административной панели, в разделе Chunks.

В разделе stats находятся опциональные настройки статистики:
mission_report_delete - удалять ли обработанные логи игры (true / false)
mission_report_backup_days - сколько дней хранить резервные копии логов игры (они хранятся упакованные в zip архив)
inactive_player_days - сколько дней игрок не должен делать вылеты чтобы статистика исключила его из рейтинга
new_tour_by_month - активация автоматической системы туров по месяцам (true / false)
win_by_score - активация расчета победы в миссии по очкам в случае если не состоялась победа по задаче
win_score_min - минимальное кол-во очков которое должна набрать коалиция чтобы сработал расчет победы по очкам
win_score_ratio - минимальное соотношение очков двух коалиций для определения победившей коалиции


В разделе email находятся настройки для отправки почты.
Настройки требуются для отправки писем активации регистрации и сброса пароля пользователя.
Мы рекомендуем использовать smtp сервер https://mailgun.com/
Их бесплатный тариф позволяет отправлять 10000 писем в месяц.
По умолчанию отправка писем отключена. Функции которые зависят от нее отключаются автоматически.


===== РЕКОМЕНДУЕМЫЕ СПОСОБЫ ВНЕСЕНИЯ ИЗМЕНЕНИЙ =====

Очки нужно изменять в административной панели сайта (http://адрес_сайта/admin/), в разделе Scoring.

Для изменения шаблонов или css стилей - рекомендуется создать копию файла(и поддиректории, если это требуется) в каталоге custom.
Файлы из custom будут иметь приоритет выше оригинальных файлов, а т.к. оригинальные файлы будут нетронуты - это облегчит обновление статистики на новую версию.

После внесения измнений в css стили, изображения, шаблоны - требуется запуск команды collectstatic для пересборки статичных файлов.
